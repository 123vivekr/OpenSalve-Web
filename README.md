# OpenSalve Web
This is the web version of Open Salve project.
Open Salve is a project to help disaster-relief efforts.

## Setup
Clone the repo.
```
git clone https://github.com/123vivekr/OpenSalve-Web.git
```
Install dependencies
```
npm install
```
